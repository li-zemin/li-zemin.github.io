1. 支持向量机

​ 本篇博文将介绍支持向量机（Support Vector Machine, SVM）算法。SVM是最好的“无需定制”的监督学习算法。为了很好地介绍支持向量机，我们首先要了解分离数据的概念以及边界问题。接下来，我们要讨论最优边界分类器，并且引出拉格朗日算子。我们也会知道核函数，有助于我们在更高维的特征空间里应用SVM算法。最后，我们会讨论SMO算法，是SVM算法的实现。

2. 边界

​ 本节内容将简单引入边界问题并讨论预测的“信任度”问题，这些概念将在第三部分正式讨论。

​
​ A距离边界最远，所以对y=1的预测更加自信。相反，C距离分类边界更近，只需要一点的变化，就可能被预测为负样本。因此，我们预测A比预测C为为正样本的信任度更大。因此，我们要找到一个边界，使我们的预测是自信的并且信任度较高。

对于核函数，可以理解为：
对于某些输入变量，我们在给定的特征空间中并不能找到合适的算法模型。因此，我们就需要把原有的输入变量映射到更高维的特征空间。所以，当我们给定核函数以后，同时会有一个对应的特征映射φ。
正常流程来讲，应该是将现有的输入变量输入通过特征映射φ映射到高维特征空间，然后再求在高维特征空间的内积（SVM算法的内积形式）。但是，特征映射往往是比较复杂的，计算复杂度较高。
核函数的计算复杂度很低。这样就可以通过核函数简化特征映射这一计算过程。

​ 现在，我们从另一个不同的角度关注核。直觉上来说，如果\(\phi(x)\)和\(\phi(z)\)彼此相近，那么\(K(x,z)=\phi(x) ^{T}\phi(z)\)会比较大。相反，如果\(\phi(x)\)和\(\phi(z)\)相距较远，或者说是正交的，那么\(K(x,z)=\phi(x) ^{T}\phi(z)\)会比较小。因此，我们可以通过K(x, z)的值衡量\(\phi(x)\)和\(\phi(z)\)之间的相似度，或者x和z之间的相似度。

x和z为两个向量，两个向量距离越相近，表示夹角越小，则内积越大。两向量正交时，内积为0。

​ 因此，我们说明了K是一个合理的核，并且对应的核矩阵K是一个对称的半正定阵。更普遍地，这是K为合理核的必要的而不是充分的条件。这是由Mercer理论推导出来的。

​ 对于Mercer定理，描述为：给定一个K表示\(\mathbb{R} ^{n}\times \mathbb{R} ^{n} \mapsto \mathbb{R}\)（也就是从两个n维向量映射到实数域）。那么K为一个有效的(Mercer)核函数的充分必要条件是，对于任意一个\(\left \{ x ^{(1)},…,x ^{(m)} \right \},(m<\infty )\)，对应的核矩阵都是对称的半正定阵。

​ Mercer定理表明为了证明K是有效的核函数，那么我们不用去寻找K，而只需要在训练集上求出各个\(K _{ij}\)，然后判断矩阵K是否是半正定（使用左上角主子式大于等于零等方法）即可。

个人理解，这里给出了一些关于寻找或者验证高斯函数是否合理的思路，具体的应用中，可能大多数情况还是用常用的几种核，比如高斯核。通过调整参数，寻找好的训练模型。

​ 核的应用不仅仅局限于SVM中。特别地，如果你有一个算法模型可以写成内积的形式，那么就可以用K(x, z)代替内积，这样就可以很神奇地将模型在更高维的控件进行训练。例如，核技巧可以被应用于感知机中以得到一个核感知算法。我们在这堂课中的好多算法都可以应用这种方法，就是我们所说的“核技巧”。
