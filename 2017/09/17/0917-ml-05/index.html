1. 支持向量机

​ 本篇博文将介绍支持向量机（Support Vector Machine, SVM）算法。SVM是最好的“无需定制”的监督学习算法。为了很好地介绍支持向量机，我们首先要了解分离数据的概念以及边界问题。接下来，我们要讨论最优边界分类器，并且引出拉格朗日算子。我们也会知道核函数，有助于我们在更高维的特征空间里应用SVM算法。最后，我们会讨论SMO算法，是SVM算法的实现。

2. 边界

​ 本节内容将简单引入边界问题并讨论预测的“信任度”问题，这些概念将在第三部分正式讨论。

​ 在下面这幅图中，x表示正训练样本，O表示负驯良样本，并且还有一个分类超平面（separating hyperplane）,三个点被标记为A,B,C,
​
​ A距离边界最远，所以对y=1的预测更加自信。相反，C距离分类边界更近，只需要一点的变化，就可能被预测为负样本。因此，我们预测A比预测C为为正样本的信任度更大。因此，我们要找到一个边界，使我们的预测是自信的并且信任度较高。
